# OpenClaw Rocket.Chat

OpenClaw 与 Rocket.Chat 的集成与部署指南。

## 内容

- [部署指南](./openclaw-rocketchat-deploy.md) - 完整的自建教程

## 快速开始

```bash
# 一键部署 Rocket.Chat
mkdir -p ~/rocket.chat && cd ~/rocket.chat
curl -sL https://raw.githubusercontent.com/rocketchat/Rocket.Chat/develop/docker-compose.yml > docker-compose.yml
docker-compose up -d
```

详见 [部署指南](./openclaw-rocketchat-deploy.md)
